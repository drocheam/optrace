8.3. Documentation#

Goal

  • website documenting library functionality and source code

  • mathematical description of methods, including images, equations and sources

  • installation instruction and simple functionality overview

Source Code

  • typing hints for almost all functions

  • docstring comments for functions and important variables

  • comments have reStructuredText syntax

Documentation

  • see documentation guidelines in Section 8.1

  • generate source code documentation using Sphinx.

  • functionality description is done using reStructuredText and embedded in the rest of the documentation

  • documentation also includes information on testing, library dependencies, changelog and documentation

  • documentation source files can be found under ./docs/source and consist of .rst files

  • run make html from the doc folder to generate the html documentation (requires the GNU make utility)

  • build files get created in ./docs/build/html, open the documentation by e.g. firefox ./docs/build/html/index.html

  • check code snippets with sphinx.ext.doctest and make doctest

Workflow

  • tox environment docs, call from main folder using tox -e docs, see the tox.ini files for details

  • this runs the following:

    • the Sphinx html documentation compilation

    • pydeps that creates a graphical representation of internal and external library dependencies

    • a bash script generating the changelog

    • a bash script generating the source file toctree

Sphinx configuration

  • see .docs/source/conf.py for details

  • Sphinx extensions

    • autodoc for automatic source code documentation generation

    • intersphinx for linking documentation of other packages

    • mathjax for LaTeX equations

    • sphinxcontrib.bibtex for a bibliography

  • author and version get automatically loaded from ./optrace/__metadata__.py

  • pyramid Sphinx theme, however we overwrite the text font family by including the ./docs/sources/_static/custom.css file. This also centrally aligns the equation label vertically.
